Introducción


Comience con Bowser Library, diseñada con SASS.Donde obtine diseños establecidos para la creación de paginas webs. El funcionamineto con CSS por sí solo puede ser divertido, pero las hojas de estilo son cada vez más grandes, más complejas y más difíciles de mantener. Aquí es donde un preprocesador puede ayudar. Sass tiene características que aún no existen en CSS , como anidamiento, mezclas, herencia y otras ventajas ingeniosas que lo ayudan a escribir CSS SÓLIDO Y FÁCIL DE MANTENER.


Bowser Library es framework de diseño web de codigo abierto, que es utlizado para la creacion de sitios webs y aplicaciones moviles con una apariencia moderna, divertida y interacctiva. Desarrollado por un grupo de desarrolladores pequeños.


Bowser Library ofrece una amplia gama de estilos y componentes predefinidos que facilitan la creación rapida y sencilla de interfaces de usuario atractivas y responsivas. Con esta libreria, los desarrolladores web pueden aprovechar un conjunto de reglas de diseño y estilos predefinidos para crear páginas web consistentes y estilizadas, lo que les permite ahorrar tiempo y esfuerzo en la creacion de diseños desde cero. Además Bowser es altamente personalizable y se integra facilmente con otros frameworks y bibliotecas, lo que lo convierte en una herramienta versátil y ampliamente utilizada en la comunidad de desarrollo web.


Acerca de SASS


Sass (Syntactically Awesome Style Sheets) es un preprocesador de CSS que permite escribir hojas de estilo de manera más eficiente y organizada. Permite el uso de variables, anidamiento de selectores, mixins, funciones, entre otras características que no están disponibles en CSS puro.


Con Sass, se puede crear un archivo .scss (o .sass) que luego se compilará en un archivo .css válido. De esta manera, se pueden mantener las ventajas de la legibilidad del código y la modularidad, sin sacrificar la compatibilidad con los navegadores.


Acerca de Bootstrap


Es uno de los frameworks de diseño web más populares y ampliamente utilizados en la industria, lo que significa que hay una gran cantidad de recursos y documentación disponible para aprender y utilizar este framework. Además, existe una amplia comunidad de desarrolladores y diseñadores que contribuyen a su desarrollo y mejora constante.

Extra extra large xxl >=1400px

Sass

Bowser Library a utilizado SASS para procesar el codigo css de toda nuestra libreria. Por lo tanto tiene un _archivo.scss (archivo parcial) con las siguientes funciones breackpoints.


              @use './variables';
              @mixin breakpoints_min($parametro){
                  @media screen and (min-width: map-get(variables.$breakpoints,$parametro)) {
                      @content;
                  }
              }
              @mixin breakpoints_max($parametro){
                  @media screen and (max-width: map-get(variables.$breakpoints,$parametro)) {
                      @content;
                  }
              }

Filas y Columnas


Bowser library trabaja el sistema de filas y columnas mediante contendores de tipo grid. Por lo cual la libreria tiene un archivo llamado _columnas-filas-grid.scss donde almacena funciones que ayudan a definir las columnas y filas. Ya que el archivo _columnas-filas-grid.scss proporciona sus funciones al archivo _grid.scss .


              
                _columnas-filas-grid.scss

                @mixin col($col){
                  grid-template-columns: $col;
                }
                @mixin row($row){
                    grid-template-rows: $row;
                }

Con lo antes mecionado en cuanto la relacion de los archivos _columnas-filas-grid.scss y _grid.scss. Tenemos que el el archivo _grid.scss contiene clases para columnas y para filas. Estas clases estan definidas por las funciones que contiene el archivo _columnas-filas-grid.scss. Bowser ha definido la cantidades de filas y columnas que puede tener un diseño. Por lo cual tendra un maximo de 4 columnas y un minimo de 1, en el caso de las filas es de la misma forma. Pero estas clases pueden ser modificadas en sus respectivos archivos. donde podra cambiar el ancho y alto que pueda tener una columna y una fila. ya que la libreria tiene definido como un tamaño estandar de 1fr.


              
                Archivo: _grid.scss
                //CLASES PARA COLUMNAS
                .-col1{.......}
                .-col2{.......}
                .-col3{.......}
                .-col4{.......}
                //CLASES PARA FILAS
                .-row1{.......}
                .-row2{.......}
                .-row3{.......}
                .-row4{.......}

Bowser library tambien establecio funciones para que un elemento de tipo grid abarque una area dentro de la cuadricula.Tambien incluye una funcion de espaciado para filas y columnas. Estas funciones pueden ser modificadas dentro del archivo _columnas-filas-grid.css . 

              
                Archivo: _columnas-filas-grid.scss

                @mixin espaciado($col,$row){
                  column-gap: $col;
                  row-gap: $row;
                }
                @mixin area-col($inicio,$final){
                    grid-column-start: $inicio;
                    grid-column-end: $final;
                }
                @mixin area-row($inicio,$final){
                    grid-row-start: $inicio;
                    grid-row-end: $final;
                }

Ahora para establecer que un contenedor sea de tipo grid, Bowser a creado una clase llamada -grid que esta en el archivo _grid.scss. Esta clase esta relacionada con un archivo llamado _ocultar-elementos.scss que contiene un funcion display.


              
                Archivo _grid.scss
                @use'../clases_utilitarias/ocultar-elementos';

                -grid{
                  @include ocultar-elementos.display(grid);
                }
